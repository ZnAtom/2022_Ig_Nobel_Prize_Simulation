# 🏆 2022 搞笑诺贝尔奖模拟项目

一个基于 Python 的社会模拟项目，模拟运气与天赋对个人财富积累的影响，以此致敬 2022 年搞笑诺贝尔经济学奖。

---

## 📋 项目简介

本项目通过计算机模拟，探讨**运气（随机事件）**与**天赋（能力值）**在个人成功和财富积累过程中的相对重要性。

模拟场景：
- 多个智能体（Person）在 100x100 的网格中随机分布
- 每个智能体拥有随机生成的能力值（服从高斯分布）
- 在时间步长内，随机位置会发生"幸运事件"或"不幸事件"
- 事件影响范围内的智能体可能获得财富增长或损失

## 🚀 快速开始

### 环境要求

- Python 3.7+
- NumPy
- Pandas
- openpyxl（用于 Excel 导出）

### 安装依赖

```bash
pip install numpy pandas openpyxl
```

### 运行模拟

```bash
python simulation.py
```

运行时会提示输入模拟参数，直接按 Enter 使用默认值：
- **模拟人数**: 默认 3000 人
- **时间步长**: 默认 40 年

---

## 📊 输出结果

模拟完成后会输出以下统计信息：

1. **财富分布统计**
   - 总财富
   - 前 20% 人口占有财富比例
   - 前 20% 人口的平均天赋值

2. **排行榜**
   - 财富排行榜前 10 名
   - 天赋排行榜前 10 名

3. **相关性分析**
   - 天赋与财富的相关系数

4. **交集分析**
   - 同时进入财富前十和天赋前十的人数

5. **数据导出**
   - 生成 `person_data.xlsx` 文件，包含所有智能体的详细数据

## 🧠 核心算法

### 能力值生成

能力值服从高斯分布：
- 均值 μ = 0.6
- 标准差 σ = 0.1
- 约束范围：(0, 1)

### 幸运事件

当幸运事件发生时，影响范围内的智能体：
```
if 能力值 > 随机数(0-1):
    财富 *= 2
    成功计数 +1
```

### 不幸事件

当不幸事件发生时，影响范围内的智能体：
```
if 能力值 < 随机数(0-1):
    财富 /= 2
    受影响计数 +1
```
